

# Update Favicon to Lekkside Logo

The browser tab currently shows the default favicon. We'll update it to use the existing Lekkside logo (`src/assets/lekkside-logo.png`).

## Steps

1. **Copy the Lekkside logo to the public directory** so it can be used as a favicon
2. **Update `index.html`** to reference the new favicon with a `<link rel="icon">` tag pointing to the copied logo

This will replace the default icon in the browser tab with the Lekkside branding.
